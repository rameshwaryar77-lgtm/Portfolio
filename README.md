# Rameshwar Nexus Portfolio

Personal portfolio website for Rameshwar Yadav, built as a static site and deployed with GitHub Pages.

## Live Links

- Live site: https://rameshwaryar77-lgtm.github.io/Portfolio/
- Repository: https://github.com/rameshwaryar77-lgtm/Portfolio

## About The Project

This portfolio presents my background as a Computer Science Engineering student specializing in Artificial Intelligence and Machine Learning. It highlights my profile, resume, interests, and contact information in a clean one-page layout.

## Sections

- Hero section
- About me
- Resume
- Interests
- Contact form with WhatsApp redirect

## Built With

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- AOS
- Swiper
- GLightbox
- Typed.js

## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/rameshwaryar77-lgtm/Portfolio.git
```

2. Open the project folder.
3. Launch `index.html` in your browser.

## Deployment

This project is deployed with GitHub Pages using the workflow in `.github/workflows/static.yml`.
